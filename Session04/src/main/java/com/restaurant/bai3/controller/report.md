Trong cách A (/bai3/orders/5), số 5 nằm trong URI Path, tức là một phần của đường dẫn
Trong cách B (/bai3/orders?id=5), số 5 nằm trong Query String, tức là phần tham số phía sau dấu ?
Theo nguyên tắc RESTful, khi cần truy cập một tài nguyên cụ thể (ví dụ đơn hàng có ID = 5), nên sử dụng Path Variable
Vì vậy em chọn Cách A vì rõ ràng, đúng chuẩn thiết kế API và dễ hiểu hơn
